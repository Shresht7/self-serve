
## ðŸ“• Reference

### `--dir`

The directory to serve.

- `Default: .` (The current directory)

### `--port`

The port to use to serve the files.

- `Default: 5327`

### `--host`

The host to use to serve the files.

- `Default: localhost`

### `--version`

Print the version number of the cli application.

- `Default: false`
